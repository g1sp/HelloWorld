# HelloWorld - World Clock

A clean, responsive webpage that displays the current time in five major cities around the world.

## Features

- **Real-time Updates**: Time updates every second
- **Five Cities**: Mumbai, London, Fremont, Singapore, and Dubai
- **Timezone Information**: Each city displays its timezone and UTC offset
- **Responsive Design**: Works seamlessly on mobile and desktop devices
- **Modern UI**: Beautiful gradient background with smooth card animations
- **Accurate Timezone Handling**: Uses browser's Intl.DateTimeFormat API for accurate timezone conversions

## Cities Included

- **Mumbai** - IST (UTC+5:30)
- **London** - GMT/BST (UTC+0/+1)
- **Fremont** - PST/PDT (UTC-8/-7)
- **Singapore** - SGT (UTC+8)
- **Dubai** - GST (UTC+4)

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/g1sp/HelloWorld.git
   ```

2. Open `index.html` in your web browser

3. The page will display the current time for all five cities and update automatically every second

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Gradients)
- Vanilla JavaScript (Intl.DateTimeFormat API)

## Browser Compatibility

Works in all modern browsers that support:
- CSS Grid
- Intl.DateTimeFormat API
- ES6 JavaScript
